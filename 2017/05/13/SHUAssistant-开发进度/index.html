SHUAssistant-开发进度

发表于 2017-05-13

SHUAssistant

中国-上海大学学生校园网代理Android APP-SHUAssistant

开发日志

2017-4-10

  • 自动登录处理方法通过验证，运作稳定，效率较高。

2017-4-11

  • 选课系统获取课表页面开始开发。

2017-4-12

  • 完成获取选课系统课表测试与验证。

2017-4-13

  • 查询课程页面开始开发。
  • UI设计提上日程。

2017-4-14

  • 查询课程任务测试数据通过。
  • 查询页面UI设计搁置。

2017-4-16

  • 完成课程查询UI

2017-4-17

  • 计算总页数与翻页功能在课程查询中实现。课程查询页面功能基本完善。
  • 修复一些小问题，简化了代码。

2017-4-18

  • 进一步完善代码容错性，针对不同的应用场景增加了判断处理。

2017-4-20

  • V1.0 ( V0.8-V1.0 ) 开始开发，从零开始。从头设计逻辑，重新规划界面，力图做到最好。
  • 作为初学安卓的开发者，一路砥砺前行，在此感谢室友SUIBIAN的帮助，为APP提出宝贵的建议，
    为界面的美化做出重大贡献。

2017-5-4

  • V0.8合并到主分支完毕，完成了课表界面的编写，由室友SUIBIAN完成图标设计和部分UI界面设计。
  • 后续开发工作正在进行

2017-5-13

  • FTP功能测试完成，通过在腾讯云的云服务器实例提供服务，可以检查，下载，安装更新。

2017-5-18

  • 自定义UI——DeleteEditText 初次尝试完成。
  • tesseract识别引擎训练数据将会自动从asset文件夹中复制到程序缓存，摆脱手动安装（终于解决了）。

2017-5-25

  • 加入了服务器检测切换功能以及当前学期，周判断（设置功能仍在开发中），但由于学校服务器切换方式诡异，以及神奇的夏季学期，估计会有不少问题。
