Everything is an Object:万物皆为对象

发表于 2017-09-26 | 分类于 java学习笔记

Java与面向对象

Although it is based on C++, Java is more of a “pure” object-oriented
language.

尽管基于C++，java相较之下更能称之为一种纯粹的面向对象的语言。

由于C++需要对C语言提供向下兼容性，这导致C++包含了许多其自身不期望的特性，这使得C++在某些方面变得过于复杂。然而Java语言确保你所做的完全是面向对象的编程，因此在开始着手于Java的学习之前应该首先将思维完全切换到面向对象的世界中去。

引用(Reference)

You treat everything as an object, using a single consistent
syntax. Although you treat everything as an object, the identifier you manipulate is actually a
“reference” to an object.

在Java中，所有的东西都应该被看作一个对象，你应该用相同的语法调用他们。也正因为你把这些东西都看作对象，你操作的实际上是这个对象的一个引用。

You might imagine a television (the object) and a remote control
(the reference). As long as you’re holding this reference, you have a connection to the
television, but when someone says, “Change the channel” or “Lower the volume,” what you’re
manipulating is the reference, which in turn modifies the object. If you want to move around the room and still control the television, you take the remote/reference with you, not the
television.

举个例子，你可以想象一个电视机(对象)和一个遥控器(引用)。你操作引用，如换台和调音量，最终导致对象的改变。每当你想换个地方坐着看电视的时候，你只要拿着遥控器(引用)而不用直接拿着电视机(对象)。
